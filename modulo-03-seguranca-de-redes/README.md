# Módulo 3: Conectar e Proteger - Redes e Segurança de Rede

Neste módulo, estudei a infraestrutura de redes de computadores, compreendendo como os dados trafegam pela internet e como implementar defesas para mitigar vulnerabilidades e ataques de rede.

## 🧠 Principais Conceitos Aprendidos

*   **Modelos de Rede (TCP/IP e OSI):** Entendimento de como as camadas de rede encapsulam e transmitem dados (foco em IP, TCP, UDP, DNS e HTTP/HTTPS).
*   **Protocolos e Portas:** Identificação das portas de comunicação mais comuns (como 80, 443, 22, 53) e suas vulnerabilidades inerentes.
*   **Arquitetura e Segurança em Nuvem:** Conceitos básicos de redes locais (LAN), redes amplas (WAN) e os fundamentos de segurança em ambientes de nuvem.
*   **Defesas de Rede:** Introdução ao funcionamento de Firewalls, VPNs e sistemas de filtragem de tráfego para proteger o perímetro de uma organização.

## 🎯 Reflexão Teórica / Atividade

Analisei o fluxo de pacotes de dados e compreendi como invasores utilizam técnicas como varredura de portas (port scanning) para mapear redes. Estudei estratégias de segmentação de rede para conter possíveis violações e garantir que dados confidenciais fiquem isolados.
